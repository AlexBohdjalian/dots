# Modular tmux setup

Previously its own repo but now part of dots.

## Getting started

Launch tmux and the plugins will be automatically loaded from the configuration.

## Keys

* Prefix: `C-s`
* Reload config: `C-s r`
* Pane navigation `C-s h/j/k/l` (vim)
