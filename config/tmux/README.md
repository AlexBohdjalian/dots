# Modular tmux setup

Previously its own repo but now part of dots.

## Getting started

Luanch tmux and install plugins with tpm: `C-s I`

You'll know it's worked because you'll have a nice theme!

## Keys

* Prefix: `C-s`
* Reload config: `C-s r`
* Pane navigation `C-s h/j/k/l` (vim)
